# Материалы Для Кейсов

Этот каталог зарезервирован под вспомогательные материалы, на которые ссылаются кейсы.

Примеры:

- скриншоты
- archived sanitized HTML
- сохранённые DOM-фрагменты
- cropped evidence images
- другие неисполняемые supporting artifacts

Не храните здесь unsafe live payloads и неконтролируемые сторонние скрипты.
