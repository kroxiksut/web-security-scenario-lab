# Trigger Phrases Cases

## Scope

Cases involving manipulative or hidden trigger instructions, prompt-like injected phrases, instruction framing, or language patterns intended to influence users or AI-assisted workflows.

## When To Use This Category

Use this category when the dominant signal family belongs to `trigger-phrases`.

If the case spans several modules equally, move it to `cross-module/` instead.

## Naming

Recommended file prefix: `TP-YYYY-NNN.md`

## Minimum Expectations

- case file written in English
- based on `../CASE-TEMPLATE.md`
- includes expected detector behavior
- includes coverage dimensions
- states whether it is already reproducible in the lab

## Suggested Future Contents

- case files
- local screenshots or references in `../assets/`
- mappings to scenario implementations
